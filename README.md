# fluoressential

Analyzing live-cell fluorescence microscopy images.

## Getting Started

### 1. Install [UV](https://docs.astral.sh/uv/getting-started/installation/)

```bash
$ curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 2. Edit Settings

fluoressential/main.py

### 3. Run Code

```bash
$ uv run main
```
